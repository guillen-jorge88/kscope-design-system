# Animations

Esta carpeta agrupa las animaciones reutilizables de la librería UI.

## Estructura
- fade: animaciones de entrada/salida por opacidad.
- slide: animaciones de desplazamiento vertical.
- scale: animaciones de entrada/salida por escala.
- rotate: animaciones de giro suave para estados de énfasis.
- collapse: animación de colapso para contenido desplegable.
- overlay: animaciones para diálogos, popovers y tooltips.
- presets: combinaciones reutilizables que reexportan animaciones base.

## Uso
Importa las animaciones desde la ruta principal de la librería o directamente desde el módulo deseado.

```ts
import { fadeAnimation, collapseAnimation } from '@kscope/ui';
```
